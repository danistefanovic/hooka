# CLI options

### --config, -c

Provide a custom path to your webhooks file.

```sh
$ hooka --config /path/to/your/webhooks/config.json
```

### --help, -h

Display help.

```sh
$ hooka --help
```

### --port, -p

Set a custom port.

```sh
$ hooka --port 1337
```

### --tlc-cert & --tls-key

Enable secure connections via HTTPS.

```sh
$ hooka --tls-cert path/to/cert.pem --tls-key path/to/key.pem
```

### --version, -v

Print Hooka version.

```sh
$ hooka --version
```
